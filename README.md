# fd_sft

## 面向故障诊断的微调数据集构建

## 第一期（2024.12）

### 数据设置

* dataset 0 —— 开源推理数据集：
  * 从GPT4的训练数据中抽取和工业或推理相关的数据
  * 共计：4222条

* dataset 1 —— CNC基础知识数据集：
  * 基于华中8型用户说明书、参数说明书、操作说明书、编程说明书抽取问答对
  * 共计：2534条

* dataset 2 —— CNC故障诊断数据集：
  * 基于故障诊断知识图谱（含正向）抽取问答对
  * 共计：2164条
